AI-Powered Talking NPCs: A New Era of Game Interaction

This is an in-depth exploration of how large language models (LLMs) can be used to revolutionize non-player character dialogue in video games. The thesis presents a layered system that integrates:

A Unity-Based Game Client
Voice Input Through Automatic Speech Recognition (ASR)
AI-Powered Dialogue Generation Using State-of-the-Art LLMs

Key Findings

1

AI-Driven NPCs Can Adapt
To Real-Time Game Contexts.

2

Voice Input Enhances
Accessibility And Engagement.

3

Qwen Model Outperformed Others In Speed And Conversational Quality.

Kieron tested models like Qwen, Mistral, and Phi to see how well they responded to player input in a real-time puzzle game. The study showed that LLMs like Qwen significantly improved immersion and dialogue quality, though challenges like slow response times and off-topic responses still exist. A voice-based interaction system was also successfully implemented, demonstrating how future games could become more lifelike and accessible.
